# 🛡️ SENTINEL-X: The Forensic Guard

**AI-Powered Transaction Forensics for Institutional Finance**

Built for JP Morgan's Kinexys Division — detecting synthetic AI entities across institutional transaction networks.

## Features

- **Kinexys Transaction Feed** — Real-time simulated transaction monitoring
- **Detective Engine** — AI entity classification (Human vs Synthetic AI) using biometric telemetry signals
- **Forensic Dossier** — High-fidelity PDF report generation for flagged transactions
- **Institutional Landing** — Secure-access entry portal with biometric authentication UX

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Lucide Icons
- jsPDF (Forensic Report Generation)

## Design System

| Token | Color | Usage |
